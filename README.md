# Job Portal Application

A full-stack Job Portal built using Spring Boot, React, MySQL, and JWT Authentication.

## Features
- User Registration & Login
- Recruiter Registration & Login
- Post Jobs
- Apply for Jobs
- Resume Upload & Download
- Recruiter Dashboard Analytics
- Job Management (Edit/Delete/Close Jobs)
- JWT Authentication & Role-Based Access

## Tech Stack
### Frontend
- React.js
- JavaScript
- CSS

### Backend
- Spring Boot
- Spring Security
- JWT

### Database
- MySQL
